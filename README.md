1. Clone the repository : git clone https://github.com/your-username/idea-diversity-meter.git
cd idea-diversity-meter
2. Create a Virtual Environment :
    # Windows
      python -m venv venv
      .\venv\Scripts\activate

    # Mac/Linux
     python3 -m venv venv
     source venv/bin/activate
3. Install Dependencies: pip install streamlit pandas numpy sentence-transformers scikit-learn plotly requests
4. Run the Application: python -m streamlit run app1.py
